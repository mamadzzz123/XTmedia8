# XT (Twitter-like) — Next.js on Vercel

برنامه شبکه اجتماعی مشابه توییتر با نام XT. بدون ایمیل/تلفن ثبت‌نام می‌کنید (فقط handle + نام نمایشی).

## دیپلوی خیلی سریع روی Vercel

1) این ریپو را روی GitHub بگذارید.
2) روی Vercel "Import Project" بزنید و همین ریپو را انتخاب کنید.
3) در تب Integrations، دو سرویس را اضافه کنید:
   - Vercel KV (برای ذخیره داده)
   - Vercel Blob (برای آپلود تصاویر)
4) Build command و output همان پیش‌فرض Next.js است. Deploy کنید.

بدون هیچ سرور پیچیده یا دیتابیس مستقل. همه چیز Serverless است.

## توسعه محلی

```bash
npm i
npm run dev
```

سپس به `http://localhost:3000` بروید.

## امکانات اصلی
- ثبت‌نام بدون ایمیل/تلفن (handle یکتا + نام نمایشی)
- ساخت پروفایل عمومی
- پست متن (تا 280 کاراکتر) + تصویر اختیاری (Vercel Blob)
- اکسپلور (آخرین پست‌های همه)
- صفحه خانه (پست‌های کسانی که دنبال می‌کنید)
- دنبال کردن/لغو دنبال
- جستجوی کاربر بر اساس handle
- تم سفید/آبی (الهام از Facebook)

## نکته
- برای جستجو، یک ایندکس ساده از handle ها در KV با کلید `user:index` نگه‌داری می‌شود.
